# Bala GitHub Portfolio

Static GitHub Pages portfolio for Balasaikrupa Puram.

## Local setup
Open `index.html` in a browser.

## Recommended repository name
`bala-portfolio` or `balasaikrupa.github.io`

## Publish using GitHub Pages
1. Create a public GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings → Pages.
4. Select branch: `main`, folder: `/root`.
5. Save.

## Custom domain
Add your domain in Settings → Pages after configuring DNS. GitHub may create a `CNAME` file automatically.
